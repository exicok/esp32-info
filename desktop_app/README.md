# 系统监控器桌面软件

## 功能
- 获取当前活动窗口的FPS
- 监控CPU使用率
- 监控GPU使用率（模拟值）
- 监控内存使用率
- 通过WiFi发送数据到ESP32
- 系统托盘图标
- 开机自启动
- 后台运行

## 运行环境
- Windows系统
- Python 3.7或更高版本

## 版本选择

### 命令行版本
- **文件**: `system_monitor.py`
- **特点**: 简单直接，适合调试和查看详细输出

### GUI版本（推荐）
- **文件**: `system_monitor_gui.py`
- **特点**: 系统托盘图标，开机自启动，后台运行

## 安装和使用（GUI版本）
1. 确保ESP32已连接到WiFi网络
2. 双击运行 `system_monitor_gui.py`
3. 首次运行会自动安装所需依赖
4. 程序会自动添加到开机自启动
5. 系统托盘会显示蓝色图标
6. 右键点击托盘图标可以：
   - 启动监控
   - 停止监控
   - 退出程序

## 配置
- **配置文件**: `config.ini`
- **默认ESP32 IP**: 192.168.1.100
- 可以手动编辑config.ini文件修改ESP32的IP地址

## 数据格式
软件通过HTTP POST请求发送数据到ESP32的`/api/system`端点，数据格式为：
- fps: 当前窗口的FPS
- cpu: CPU使用率（百分比）
- gpu: GPU使用率（百分比）
- mem: 内存使用率（百分比）

## 注意事项
- ESP32和电脑需要在同一WiFi网络中
- 首次运行会自动安装依赖包，可能需要一些时间
- 程序会在后台运行，通过系统托盘图标控制
- 开机后会自动启动并开始监控
